# VueSIP Overview
- Purpose: Headless Vue 3 composable library for building SIP/WebRTC softphone interfaces.
- Key capabilities: SIP connection management, call handling, DTMF, audio device selection, playground demos, extensive docs.
- Core tech: Vue 3 Composition API, TypeScript, Vite build tooling, JsSIP/WebRTC integrations.
- Repo structure highlights: `src/` library code (composables, core, providers, plugins, stores, utils, types); `tests/` with unit/integration/e2e suites (Playwright, Vitest); `docs/` VitePress site; `playground/` demo app; `examples/` production-ready samples.
- Tooling: TypeScript strict configs, ESLint + Prettier, Husky + lint-staged, Vitest, Playwright, TypeDoc, VitePress.
